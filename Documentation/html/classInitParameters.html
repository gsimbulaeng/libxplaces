Detailed Description

InitParameters is a container for parsed xPlaces Init parameters.

An InitParameters contains init parameters for node and xpNetwork initialization, obtained by parsing command-line parameters. The correct syntax for the method is: 

  [PATH_TO_EXECUTABLE]/[EXECUTABLE_NAME] [OPTIONS]

The accepted options are: 

  -p [INTEGER] The port used by the node
  -s [http://ip:port] The address of the server
  -i [INTEGER] The announce interval in seconds
  -t [STRING] OPTIONAL: The node type
  -n [STRING] The node name
