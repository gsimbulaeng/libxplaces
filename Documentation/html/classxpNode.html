Detailed Description

A template for an xplaces network endpoint.

An xpNode is every object in the xplaces network that has an xpDescriptor and could send announce messages through the network using UDP. It's a superclass for xpDevice and xpApplication and it defines that every extended class must have an xpDescriptor as container for network informations as ip, port, name and node type.
