Detailed Description

xpDescriptor is a container for network informations.

An xpDescriptor has to be instanced by every node in the xplaces network that has to connect. Inheriting from the xpMessage superclass the methods to obtain address and session, an xpDescriptor must be instanced by specifying the name, the type and the properties of the node.
