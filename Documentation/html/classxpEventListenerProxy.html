Detailed Description

xpEventListenerProxy is an proxy to an xpEventListenerImpl, defined by a properties list that brings informations concerning event listeners.

An xpEventListenerProxy is a event listener proxy to an xpEventListenerImpl, registered on a xpDeviceImpl, that forwards events, emitted from related xpDeviceImpl, through the network. The class has to implements the pure virtual methods of the superclass.
